# **Upgrading the sleeping brain with targeted memory reactivation** 

**Author(s):** [Delphine Oudiette and Ken A. Paller] | **Year:** [2013]



**Reason for selection**
This paper serves as the foundational conceptual model for the Targeted Memory Reactivation (TMR) system. It maps the theoretical boundaries and biological mechanisms required to understand how external cues interact with the sleeping brain.
